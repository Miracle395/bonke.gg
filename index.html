About Bonke


Bonke is a community focused cryptocurrency created by one of the original Dogecoin shibes from 2013. It was created for one purpose, to create a new and fun community just like the original Dogecoin community.




Simple Layer 1

Bonke is a layer 1 blockchain, not an ERC20 token. There are no confusing smart contracts that can be used by bad actors to steal funds.



Born From Dogecoin

Bonke is a fork of Dogecoin. Developers and users familiar with Dogecoin will feel right at home with Bonke.


Secure Blockchain

Like Dogecoin, Bonke's blockchain is secured using Merged Mining.


Completely Safe

Bonke is completely open source. You can view the source code here.
